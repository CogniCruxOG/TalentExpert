# Content Sourcing & Decisions Log

This site was rebuilt from three inputs:

1. **Reference HTML** — used **only** for design language (palette, typography, animations, interactions, responsive behaviour). Its text was placeholder and was **not** used as final content.
2. **Website Content Blueprint (v1.0)** — primary source for all copy, site map, SEO, forms, footer.
3. **Corrections document** — the client's latest review. **Where it differs from the Blueprint, Corrections win.**
